Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content to the user
  • The body can be structured with semantic element such as header, main and footer
  • attributes qualifies elements
  • Elements are often assimilated with tags however they are slightly different

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • we can use inline CSS, Internal Style Sheet or External Style Sheet
  • Inline CSS applies to a specific html tag
  • Internal Style Sheet contains CSS rules for the webpage in the head section of the HTML
  • External Style Sheet needs to be linked to the HTML
  • External Style Sheet respects the Separation of Concerns approach
  • CSS is built with selector, property and value
  • DRY code: Do Not Repeat Yourself: Use comma to apply CSS to multiple selectors
  • CSS attributes: Class and id

GIT

  • git clone "git hub repository ssh": Clone a git hub repository on local directory
  • git checkout branch: Switch to branch
  • git status: Check which branch you are in
  • git add -A : Stage changes before commit
  • git commit -m "name of change": Commit changes
  • git pull origin main: Pull latest version of github repository on local directory
